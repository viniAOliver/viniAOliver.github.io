Visão Geral

X-TF-IDF é uma ferramenta desenvolvida em Python para coleta e análise de postagens do X/Twitter. Combinando técnicas de web scraping e acesso à API oficial, ela extrai tweets e aplica o modelo TF-IDF (Frequência de Termo - Frequência Inversa em Documentos) para avaliar a relevância dos termos em um conjunto de dados.

Ideal para análise de tendências, mineração de opiniões e experimentos com vetorização em Processamento de Linguagem Natural (PLN), o X-TF-IDF oferece uma implementação prática e extensível, com suporte completo via linha de comando e uma interface web para testes e buscas. O projeto é 100% Python, utilizando bibliotecas populares como requests, nltk, tweepy, e é totalmente open-source.
