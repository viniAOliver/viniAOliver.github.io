X-TF-IDF

X-TF-IDF é uma ferramenta baseada em Python para coleta e análise de postagens do Twitter. Ela utiliza web scraping com acesso à API para extrair tweets do X/Twitter e aplica o modelo TF-IDF (Frequência de Termo - Frequência Inversa em Documentos) para avaliar a relevância dos termos no conjunto de dados coletado. Esta ferramenta é ideal para mineração de opiniões e experimentos de vetorização em PLN (Processamento de Linguagem Natural).
